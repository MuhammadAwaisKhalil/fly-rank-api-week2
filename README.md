<h1>Database Introduction for Data Persistence</h1>
<h3>SQLite Database</h3>
<p>Sqlite database was chosen because it is light weight and easy to use without any prior installatiosn requied
It represents a 1 tier architecture where database resides locally on the machine using it
Suitable for tasks like this one.</p>
<br>
The database is started using the startup event handler of fast api that as soon as app laumches database connection is established
